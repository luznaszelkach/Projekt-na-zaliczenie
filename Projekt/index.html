Do czego używana jest Java?

Java jest używana w wielu aplikacjach, których możesz używać na co dzień. Od naukowych aplikacji, finansowych do gier.



Pierwszy Program w Javie

Tradycyjnie pierwszy program powinien wyświetlać tekst Hello World w konsoli. Do utworzenia tak prostego programu możemy użyć notatnika. Wystarczy przepiać następujący kod:

Kod:
public class PierwszyProgram{
public static void main(String []args){
System.out.print(”Hello World”);
}
}

Notatnik należy zapisać z rozszerzeniem .java zamiast .txt. Nazwę pliku musimy zostawić taką sama jak klasy, czyli PierwszyProgram.java (wielkość liter ma znaczenie).

Uruchomienie Programu

Otwieramy konsole (cmd), a następnie przechodzimy do folderu z naszym notatnikiem. Plik musimy skompilować wpisując javac PierwszyProgram.java

Wzór
javac NazwaKlasy.java

Dzięki temu utworzy nam się plik z rozszerzeniem .class w środku, którego znajduje się bytecode.
Teraz możemy uruchomić program wpisując java PierwszyProgram.

Wzór
java NazwaKlasy

Program powinien wydrukować na ekranie napis Hello Wrold.



Często używane Pojęcia w Javie

Klasa

Jest szablonem definiującym postać obiektu. Określa ona zarówno dane obiektu, jak i kod, który działa na tych danych.

Obiekt

Jest to instancja klasy, inaczej jej reprezentacja i nośnik. Jeżeli mówimy o czymś, że jest obiektem w Javie, to tak naprawdę wyobrażamy sobie całą klasę i wszystko to co się w niej znajduje. Poprzez obiekt możemy odwoływać się do zmiennych, metod i innych elementów w niej zawartych.

Metoda

Inaczej nazywana funkcją. Dokonuje się w niej różnych operacji. Metody mogą nie zwracać nic, ale równie dobrze dać w wyniku typ prosty, lub zwrócić obiekt. Niektóre mogą przyjmować także parametry, inne działać tylko na rzecz obiektu, który je wywołuje.

Dziedziczenie

Mechanizm, dzięki któremu jedna klasa może odziedziczyć zachowanie i atrybuty po innej klasie. Klasa, która dziedziczy po innej klasie, nazywana jest podklasą. Klasa, po której się dziedziczy, nazywa się klasą bazową lub nadklasą. Klasa może posiadać tylko jedną nadklasę i wiele podklas. Podklasa dziedziczy wszystkie atrybuty i całe zachowanie po klasie bazowej.



Zmienne



Jak to przechować?!

Zmienna jest to element, w którym możemy przechowywać wartości, a następnie odwołać się do nich po nazwie.

Literał

Poza zmiennymi w instrukcji programów Java pojawiają się literały. Literał to dowolna liczba, tekst lub inna informacja, która w sposób bezpośredni reprezentuje wartość.


W procesie tworzenia zmiennej wyróżniamy dwa etapy:

  • deklaracja
  • inicjalizacja

Wzór
Typ nazwa = wartość;

Przykład
Int zmienna;
zmienna = 13;

Typy podstawowe (ang. primitive types)

    Liczby całkowite

  • byte
  • short
  • int
  • long

    • Liczby zmiennopozycyjne

  • float
  • double

    • Typ znakowy

  • char

    • Typ boolowski

  • boolean

Stałe

Zmienna z kluczowym słowem final oznacza, że można tylko jeden raz przypisać wartość i nie będzie już jej można zmienić w programie.


Łańcuchy

W Javie istnieje standardowa biblioteka zawierająca predefiniowaną klasę o nazwie String. Każdy łańcuch w cudzysłowie jest obiektem klasy String.



Wzór
String nazwa = ”tekst”;

Przykład
String tekst = ”test”;


Instrukcje warunkowe



Pouczmy się!

Instrukcje warunkowe pozwalają na wykonanie bloku instrukcji, jeśli zostanie spełniony warunek, w przeciwnym wypadku wykonane są instrukcje z bloku else.

Wzór
If (warunek) {
instrukcja
} else{
Instrukcja
}

Przykład
If(cena>10){
Koszyk+=10;
}else{
Koszyk+=cena;
}



Pętle



To takie proste!

while

Pętla while wykonuje instrukcje (albo blok instrukcji) tak długo, jak długo warunek ma wartość true. Warunek jest sprawdzany na samym początku działania, co oznacza, że pętla może się wcale nie wykonać.

Wzór:
while (warunek) {
instrukcje
}

do while

Pętla do while pierwsze wykonuje instrukcje, potem sprawdza warunek. Wykona się co najmniej raz.

Wzór:
do {
instrukcje
} while (warunek)

for

Pętla for to pętla o określonej liczbie powtórzeń. Liczba iteracji instrukcji for jest kontrolowana za pomocą licznika lub jakieś innej zmiennej, której wartość zmienia się po każdym powtórzeniu.

Przykład/Wzór
for ( int i = 0; i<10; i++){
instrukcje
}
